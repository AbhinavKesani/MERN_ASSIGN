Week 4 – Simple E-Commerce Backend Application

This week focuses on developing a backend application for a simple e-commerce system using Node.js, Express.js, and MongoDB. The project helps in understanding how real-world backend systems manage users, products, and database operations in an online shopping platform.

The application was designed using RESTful API architecture and includes CRUD functionality for managing users and products. MongoDB was integrated as the database, while Mongoose was used for schema modeling and data validation.

 Project Overview

The Simple E-Commerce Backend project simulates the backend workflow of an online shopping platform. It provides APIs to manage users and products while storing data in a MongoDB database.

The project focuses on:

Backend API development
Database integration
Routing and middleware
CRUD operations
Structured project architecture
 Features Implemented
 User API

Implemented APIs to manage user-related operations such as:

Creating users
Fetching user details
Updating user information
Deleting users

These APIs help in handling user management functionality within the application.

 Product API

Developed product management APIs to:

Add products
Retrieve product details
Update product information
Delete products

This functionality forms the core of the e-commerce backend system.

 CRUD Operations

Implemented complete CRUD operations for both users and products.

Operation	Purpose
Create	Add new records
Read	Fetch existing data
Update	Modify stored data
Delete	Remove records

CRUD functionality improves understanding of backend data management.

▶ How to Run the Project

Make sure Node.js and MongoDB are installed on your system.

Install project dependencies:

npm install

Start the backend server:

npm start
