# Week 1 — JavaScript Fundamentals

This folder contains small JavaScript exercises covering basic language features and common operations.

Contents
- `Array-operations-simple1.js` — basic array creation and access
- `Array-operations-simple2.js` — iteration and simple transforms
- `Array-operations-simple3.js` — common array helpers
- `Array-operations-advanced1.js` — advanced array algorithms
- `Array-operations-advanced2.js` — higher-order functions and chaining
- `Array-operations-advanced3.js` — performance-aware array handling
- `Array-operations-advanced4.js` — complex array problems
- `Array-operations-advanced5.js` — challenge problems
- `Control-statements1.js` — `if`, `else`, and switch examples
- `Control-statements2.js` — loops: `for`, `while`, `do..while`
- `Control-statements3.js` — loop control and patterns
- `Object-operations-simple1.js` — object creation and property access
- `Object-operations-simple2.js` — object iteration and methods
- `Object-operations-simple3.js` — shallow/deep updates and cloning
- `Operators.js` — examples of arithmetic, logical, and comparison operators

Quick start
- Prerequisites: Node.js (v14+) installed.
- To run an example, open a terminal in this folder and run:

```bash
node Array-operations-simple1.js
```

Notes
- Each file is a standalone script demonstrating a focused concept. Modify and run them to experiment.
- For deep-copy and cloning exercises, check related files in other weeks (e.g., `week2`).
