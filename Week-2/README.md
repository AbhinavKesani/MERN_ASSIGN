## Week 2 — JavaScript Practice & Small Projects

This folder contains focused exercises and small project skeletons that build on Week 1 concepts.

Contents
- `Date_hands-ons.js` — working with the `Date` API and time calculations
- `Shallow & Deepcopy1.js` — shallow vs deep copy examples and pitfalls
- `Shallow & Deepcopy2.js` — utilities and patterns for cloning objects
- `onlineLearningPlatform/` — mini project: course catalog engine and data
- `ShoppingCartSystem/` — mini project: cart logic and sample data
- `todo-app/` — simple todo app starter

Quick start
- Prerequisites: Node.js (v14+) installed.
- To run a script from this folder, open a terminal here and run, for example:

```bash
node Date_hands-ons.js
```

Project notes
- The `onlineLearningPlatform` contains `courseCatalogEngine.js` and supporting data — use it to explore data modeling patterns.
- The mini projects are lightweight starters; open their folders for README or run instructions inside.

Tips
- Start with the `Shallow & Deepcopy` examples to understand object mutation and cloning.
- Use `node --inspect` or browser devtools for debugging small apps in the subfolders.
