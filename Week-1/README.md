- Operators

## Topics Covered
Week 1 – JavaScript Fundamentals and Core Programming Concepts

This week focuses on building a strong foundation in JavaScript by practicing essential programming concepts and problem-solving techniques. The exercises cover basic syntax, operators, decision-making statements, loops, arrays, objects, and functions. These programs help in understanding how JavaScript works internally and improve logical thinking for real-world application development.

The practice files included in this week are designed to strengthen both theoretical understanding and coding skills through hands-on implementation.

 Topics Covered
 Arithmetic and Logical Operators

Learned and implemented different types of operators used in JavaScript, including:

Arithmetic Operators (+, -, *, /, %)
Assignment Operators
Comparison Operators
Logical Operators (&&, ||, !)
Increment and Decrement Operators

These programs help in understanding calculations, conditions, and expression evaluation.

 Conditional Statements

Practiced decision-making concepts using:

if
if...else
else if
switch statements

These statements are used to execute different blocks of code based on conditions.

 Loops and Iteration

Implemented looping concepts to perform repetitive tasks efficiently using:

for loop
while loop
do...while loop

Loop-based programs improve problem-solving and iteration logic.

 Array Operations

Worked with arrays to perform various operations such as:

Adding and removing elements
Traversing arrays
Searching values
Sorting and reversing
Using array methods like:
push()
pop()
shift()
unshift()
map()
filter()
reduce()

Both simple and advanced array manipulation programs were practiced.

 Object Operations

Learned how to create and manipulate JavaScript objects, including:

Accessing object properties
Updating values
Adding and deleting properties
Iterating through objects
Using nested objects

These concepts are important for handling structured and real-world data.

 Functions

Implemented reusable blocks of code using:

Function declarations
Function expressions
Arrow functions
Parameter passing
Return statements

Functions help in writing modular and maintainable code.
